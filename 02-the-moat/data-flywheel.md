Correction Loop – 2/5

What you capture today:
User feedback on reports, accepted/rejected recommendations, follow-up questions, dashboard interactions.

How it compounds:
The AI learns which insights are useful, which alerts are ignored, and which recommendations lead to action. However, there is no clear automated retraining loop yet.

Preference Loop – 3/5

What you capture today:
Favorite KPIs, reporting frequency, preferred dashboards, alert thresholds, business goals, executive vs operational views.

How it compounds:
Over time Menza delivers increasingly relevant insights and executive summaries tailored to each user and team.

Domain Context Loop – 4/5

What you capture today:
Cross-functional business data from ecommerce, CRM, marketing, finance, operations, customer support, and sales systems.

How it compounds:
Insights generated in one domain improve understanding in others. For example, marketing spend can be connected to revenue trends, customer retention, inventory, and support tickets.

Network Loop – 1/5

What you capture today:
Limited aggregation across customers.

How it compounds:
Currently, one customer's usage does not significantly improve outcomes for another customer. There are weak network effects.
